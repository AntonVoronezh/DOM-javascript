Функция как переменная

Задача 1. Сделайте функцию func, которая будет возвращать через return какую-либо строку.

Задача 2. ыведите результат работы функции func алертом на экран.

Задача 3. Выведите исходный код функции func алертом на экран.

Задача 4. Запишите в переменную func число 123, тем самым затерев функцию из этой переменной. Выведите новое значение переменной func на экран.

Задача 5. Сделайте функцию func1, которая будет возвращать через return число 3.

Задача 6. Скопируйте исходный код функции func1, в переменную func2.

Задача 7. Выведите на экран сумму результатов работы функций func1 и func2.

Задача 8. Сделайте безымянную функцию, которая будет возвращать через return число 1. Запишите эту функцию в переменную func1.

Задача 9. Сделайте безымянную функцию, которая будет возвращать через return число 2. Запишите эту функцию в переменную func2.
