Задачи на продвинутую работу с функциями JavaScript

http://theory.phphtml.net/courses/javascript/practice/1/h.html

Задача 1. Сделайте функцию detElem, которая параметром будет принимать массив и какое значение и возвращать массив, в котором все элементы с таким значением будут удалены.

Задача 2. Сделайте функцию getDiff, которая параметром будет принимать два массива и возвращать массив элементов, которые не присутствуют одновременно в обоих массивах

Задача 3. Сделайте функцию arrayRand, которая параметром будет принимать массив и возвращать случайный элемент из этого массива.

Задача 4. Сделайте функцию uniq, которая будет оставлять в массиве только уникальные элементы (то есть будет удалять дубли).

Задача 5. Сделайте функцию shuffle, которая параметром будет принимать массив и перемешивать его элементы в случайном порядке.

Задача 6. Сделайте функцию, которая параметром будет принимать число, а возвращать это же число прописью. Например, число 115 прописью это сто пятнадцать. Пусть функция работает с числами до миллиона
