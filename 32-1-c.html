Задача 2. Дана строка. Сделайте заглавным первый символ каждого слова этой строки. Для этого сделайте вспомогательную функцию ucfirst, которая будет получать строку, делать первый символ этой строки заглавным и возвращать обратно строку с заглавной первой буквой

Задача 4. Сделайте функцию isSimple, которая параметром будет принимать целое число и возвращать true, если число простое и false - если не простое. Простое число - это такое число, которое не делится ни на какое другое, кроме кроме себя самого и единицы. Например, число 31 - простое. То есть в нашем случае нужно проверить, что число 31 не делится на все числа от 2 до 30.

Задача 5. С помощью функцию isSimple из предыдущей задачи выведите на экран все простые числа из промежутка от 1 до 100.

Задача 6. Сделайте функцию, которая возвращает множественное или единственное число существительного. Пример: 1 яблоко, 2 (3, 4) яблока, 5 яблок. Функция первым параметром принимает число, а следующие 3 параметра — форма для единственного числа, для чисел два, три, четыре и для чисел, больших четырех, например, func(3, 'яблоко', 'яблока', 'яблок').
